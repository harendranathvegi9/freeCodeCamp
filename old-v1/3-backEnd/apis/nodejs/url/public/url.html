API URL Shortener

    1) You submit a URL as a parameter and receive a shortened URL in JSON response.
    2) If you pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
    3) If you visit a shortened URL you will be redirected to the original link.
Example Creation
https://brusbilis.com/freecodecamp/5-api/url/new/https://www.freecodecamp.com
https://brusbilis.com/freecodecamp/5-api/url/new/https://freecodecamp.com
Creation JSON response
{ "original": "http://freecodecamp.com" , "short": "https://brusbilis.com/5-api/url/1596" }
Usage
https://brusbilis.com/5-api/url/1596 redirects to https://freecodecamp.com
