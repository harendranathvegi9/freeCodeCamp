API Image Search

    1) You can get the image URLs, alt text and page urls for a set of images relating to a given search string.
    https://brusbilis.com/freecodecamp/old-v1/apis/image/api/search?q=freecodecamp
    2) You can paginate through the responses by adding a ?offset=2 parameter to the URL.
    https://brusbilis.com/freecodecamp/old-v1/apis/image/api/search?q=board games&num=10&offset=5
    3) You can get a list of the most recently submitted search strings.
    https://brusbilis.com/freecodecamp/old-v1/apis/image/api/recent
GET /api/search?=q[&num=number][&offset=number]
