API URL Shortener Docs

    1) You submit a URL as a parameter and receive a shortened URL in JSON response.
    2) If you pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
    3) If you visit a shortened URL you will be redirected to the original link.

Creation Example :

https://brusbilis.com/freecodecamp/old-v1/apis/v1/url/new/https://www.freecodecamp.com https://brusbilis.com/freecodecamp/old-v1/apis/v1/url/new/https://freecodecamp.com

Creation JSON response :

{ 
	"original": "https://brusbilis.com/freecodecamp/old-v1/apis/v1/url/new/http://freecodecamp.com" , 
	"short": "https://brusbilis.com/freecodecamp/old-v1/apis/v1/url/168" 
}

Usage

https://brusbilis.com/freecodecamp/old-v1/apis/v1/url/168
redirects to
