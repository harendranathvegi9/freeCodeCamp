API Timestamp Docs

    1) You pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (Month Day, Year | Month Day Year)
    2) If it does, it returns both the Unix timestamp and the natural language form of that date.
    3) If it does not contain a date or Unix timestamp, it returns null for those properties.

Example

https://brusbilis.com/freecodecamp/5-api/timestamp/July%2024,%201995
https://brusbilis.com/freecodecamp/5-api/timestamp/July 24 1995
https://brusbilis.com/freecodecamp/5-api/timestamp/806536800
Output:
{ "unix": 806536800, "natural": "July 24, 1995" }
